    Module 3 Assignment

    In this module, you are going to be exposed to a commonly used system, jsfiddle.  Jsfiddle allows you to work with HTML, CSS and javascript all in a single page which can save time when experimenting with code.  By creating an account on the website, you can also delete "fiddles" which are these tests.

    Using the jsfiddle system, complete the items below:

    Use a CSS Template

    There are many free CSS templates on the Internet. For example, http://www.free-css-templates.com/  has many free CSS templates.  Please download a favorite CSS template and use it to build a small-scale website with at least two web pages. You can use any content you like for the web pages. You will link from the page you build with jsfiddle to your original website using an anchor link ("A" tag) to create a two page website.  One page is created in jsfiddle and the other is your page from homework #1.

    Please add a hyperlink CSS Template Example on your personal website to point to this site.

    Use JavaScript

    Create a web page with JavaScript to do the following.  These can all be on one page. 

    1.) Prompt the user for their name.

    2.) Use a pop-up box to welcome the user by name.

    3.) Display the current date on the page in the following format: October 30, 2000.  Do not display the time.  Do not "hard code" the date; if I load the page tomorrow, I should get a different date than if I load it today.

    4.) Display the last modified date of the document.

    5.) Display a message saying Good Morning if it is in the morning, Good Afternoon if it is in the afternoon, and Good Evening if it is in the Evening.

    Hint: The Internet search engine will be helpful for you to complete this JavaScript assignment.

    Please add a hyperlink Javascript Example on your personal website to point to this JavaScript assignment page.

     

    Please submit your websites URL through the assignment feature of the Canvas system.
